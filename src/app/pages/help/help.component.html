How to use

Core Keeper Save Editor is a tool that allows you to easily modify your character save files. Currently, it is possible to modify the Inventory, Skills and Talents, Unlocked Souls and Character Information


Importing save data

In order to modify an existing character, you first need to import it into the application. Click the Import button in the top menu and choose the save file you want to edit.

Check this guide if you are not sure where to find Core Keeper's save-files. Character save-file are found in the /saves directory and are labeled <index>.json. For example, if you want to import the character in the second character-slot, you need to selected 1.json (index starts at 0).


Exporting save data

To export your modified save-file and load it in the game, click the Export button in the top menu and overwrite the original .json file. We highly recommend creating a backup of your original save-file before this step in order to avoid any potential losses.

Because of the way, the game loads character save-date, the game needs to be restarted to load the modified save-file. To make sure the save-file will be loaded correctly, close the game before exporting the save-file and restart it afterward.
