How to Use

Core Keeper Save Editor is a tool that allows you to easily modify your character save files. Currently, it is possible to modify the Inventory, Skills and Talents, Unlocked Souls and Character Information.


Importing Save Data

In order to modify an existing character, you first need to import the save file of the character into the application. Click the Import button in the top menu and choose the save file you want to edit.

Check out this guide, if you are not sure where to find Core Keeper's save files. Character save files are found in the /saves directory and are labeled <index>.json. For example, if you want import the character from the second character slot, you need to select 1.json (index starts at 0).


Exporting Save Data

To export your modified save file and load it into the game, click the Export button in top menu and overwrite the original .json file. We highly recommend creating a backup of your original save file before this step in order to avoid any potential losses.

Because of the way the game loads character save-data, the game needs to be restarted to load modified save files. To make sure save files will be loaded correctly, close the game before exporting and importing save files and restart it afterward.
